# Emos签到助手

自动签到 [Emos](https://emos.best) 站点，追踪萝卜收益，查看签到历史。

## 功能特性

- 🔄 **自动签到** — 支持 Cron 定时签到，默认每天 8:00 执行
- 🥕 **萝卜追踪** — 自动统计累计签到次数和萝卜收益
- 📊 **签到历史** — 可视化查看签到记录，支持桌面端表格和移动端卡片
- 🔔 **通知推送** — 签到结果通过系统通知渠道推送（站内信 / 微信 / Telegram 等）
- 🛡️ **三重幂等保护** — 本地日期 + API sign 字段 + 线程锁，防止重复签到
- 🎨 **Vue 前端** — 侧栏全页入口，签到状态 + 签到历史双页面
- 📝 **一言附言** — 支持从 [uapis.cn](https://uapis.cn) 获取一言作为签到附言

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用插件 | 开关 | 关 |
| 接口地址 | Emos API 地址 | `https://emos.best` |
| 密钥 | Bearer Token，格式 `xx_xxx`，从 Emos 主站获取 | 空 |
| 签到时间 | 5 位 Cron 表达式 | `0 8 * * *` |
| 签到附言 | 可选，随机签到可获得更多萝卜 | 空 |
| 签到通知 | 签到结果推送通知 | 开 |
| 立即执行一次 | 保存后立即跑一次签到 | 关 |
| 左侧导航入口 | 在侧栏显示插件入口 | 开 |
| 启用一言附言 | 签到附言使用一言(随机句子)API | 关 |
| 一言 API 地址 | 一言(随机句子)API 接口地址 | https://uapis.cn |

## 获取密钥

1. 访问 [Emos](https://emos.best) 并登录
2. 在个人设置中找到「密钥」，格式为 `xx_xxx`
3. 复制填入插件的「密钥」配置项即可

## 注意事项

- ⚠️ **禁止将签到时间设为 0:00（凌晨）**，否则可能导致账号被限制
- ⚠️ **禁止频繁轮询 API**，插件已内置幂等保护，每天只签到一次
- 如遇 CF 403，可尝试将接口地址改为备用地址 `https://api.emos.best`
